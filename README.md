# SPRING.VOTE

This is a project created for an assignment for Advanced Java Frameworks.

The project consists in a voting system, where it allows the user to register voters and add a single vote for each voter. There is also a page for seeing graphs related to the data stored in the app.

To run this application, run this project as a Spring Boot application. It will run a local server in the port 8080. Access the app with the URL `localhost.com:8080`.
